15. Mechanics Grade 11 Please give the correct answe, don;t copy paste


Jawab:

KE gain = 4032000 J

PE loss = 42000000 J sinθ

(ii)

θ = 3.3°

Penjelasan dengan langkah-langkah:

(i)

KE gain = 1/2 * mv^2

= 1/2 * 14000 * 24^2

= 4032000 J

PE loss = mg × ABsinθ

= 14000 * 10 * 300sinθ

= 42000000 J sinθ

(ii)

Work Done by Driving Force = KE gain – PE loss   + Work Done by resistance

5000 = 4032 – 42000sin θ + 3360

θ = 3.3°
