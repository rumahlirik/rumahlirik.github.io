Korban tewas pada umumnya karena tertimpa bangunan yang roboh. Sementara itu korban luka-luka banyak terjadi karena kepanikan yang luar biasa.

16. modal per. 31 des 2006 = 18.750.00 modal per. 1 jan 2005 = 20.000.000 laba tahun buku 2006 = 2.000.000 prive tahun 2006?


Modal akhir = Modal awal + Laba - Prive

maka.
jika prive = P

18.750.000 = 20.000.000 + 2.000.000 - P

18.750.000 = 22.000.000 + P
P = 22.000.000 - 18.750.000
P = 3.250.000

jadi prive nya = 3.250.000modal awal                             = 18750000
laba bersih           = 2000000
prive                    = x
kenaikan modal                       = 2000000 - x
modal akhir                             = 20000000

persamaan : ( misalkan x = prive )
18750000 + 2000000 - x = 20000000
                                - x = 20000000 - 18750000 - 2000000
                                - x = - 750000
                                  x = - 750000 : ( - 1 )
                                  x = 750000
Jadi prive sebesar 750000
