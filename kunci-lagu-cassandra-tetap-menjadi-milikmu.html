Willingness berati kesediaan, kemauan untuk melakukan sesuatu. Untuk menyatakan itu Kita dapat menggunakan modal will/would yang berati akan/Mau/sedia. Will Dan Would memiliki kesamaan. Perbedaannys adalah will digunakan untuk bentuk present tense ( sekarang ) Dan would untuk masa lampau. Untuk membuat kalimat dalam "will" Bahasa inggris juga harus memperhatikan pola/susunan kalimatnya , yaitu :

"will" untuk menyatakan kemauan dimasa sekarang (present)

Pattern

(+) S + will+ Verb 1 + object

(-) S + will/would + not + Verb 1 + object

(?) will + S + Verb 1 + object
