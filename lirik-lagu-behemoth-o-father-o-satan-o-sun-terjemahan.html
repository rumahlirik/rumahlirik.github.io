16. O Frio ThuO WedO MonO SunO TuesSederhanakan Bentuk acar berikutg√2 + √72-√578Joe up on you ​


Jawaban:

-10√2

Penjelasan dengan langkah-langkah:

= √2 + √72 - √578

= 1√2 + √36 x 2 - √289 x 2

= 1√2 + √6² x 2 - √17² x 2

= 1√2 + 6√2 - 17√2

= (1 + 6 - 17)√2

= (7 - 17)√2

= -10√2
