4. Seorang pedagang menjual tv sg harga 1.400.000 .tentukan harga beli tv tersebut jika pedagang mendapat rugi 10%


Kerugian = Harga jual x banyaknya persentase kerugian
                 = Rp. 1.400.000 x 10%
                 = Rp. 1.400.000 x 10/100
                 = Rp. 14.000.000/100
                 = Rp. 140.000

Harga beli = Harga jual + Kerugian
                   = Rp. 1.400.000 + Rp. 140.000
                   = Rp. 1.540.000

32. Pak sultan membeli mobil sg harga 150000000.setelah 3 bln dipkai mobil tsb dijual sg harga 136500000.persentse kerugian yg ditanggung pak sultan adalah


Jawab:

9%

Penjelasan dengan langkah-langkah:

harga awal : 150.000.000

harga jual : 136.500.000

rugi = harga jual-harga awal

= 136.500.000-150.000.000= -13.500.000 (minus artinya rugi)

persentase kerugian = banyak rugi/harga awal x 100% = 13.500.000/150.000.000 x 100% = 9%

Jadi kerugian yg ditanggung pak sultan sebesar 9%
