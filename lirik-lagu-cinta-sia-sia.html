31. jika umur tono+umur cinta=80,umur tono + umur kesia =98, umur cinta+ umur kesia =94 berapa umur tono kesia dan cinta


Misalkan :
Umur tono adalah a
Umur cinta adalah b
Umur kesia adalah c

Maka :
a + b = 80 → persamaan 1
a + c = 98 → persamaan 2
b + c = 94 → persamaan 3

eleminasi persamaan 1 dan 2 :
a + b = 80
a + c = 98
-------------- -
   b - c = -18 → persamaan 4

eleminasi persamaan 4 dan 3 :
b - c = -18
b + c = 94
-------------- +
    2b = 76
      b = 76 : 2
      b = 38

Substitusikan nilai b ke persamaan 1:
a + b = 80
a + 38 = 80
a = 80 - 38
a = 42

Substitusikan nilai a ke persamaan 2 :
a + c = 98
42 + c = 98
c = 98 - 42
c = 56

Jadi, Umur tono = 42 tahun, umur cinta = 38 tahun dan umur kesia = 56 tahun
