15. Jelas 5 Kata kunci Dari keyboard


Jawaban:

Ctrl + C berfungsi untuk melakukan copy pada obyek yang dipilih (memilih obyek, fungsinya hampir sama seperti Ctrl + X namun shortcut ini berfungsi untuk memilih).

Ctrl + S berfungsi untuk melakukan save pada program yang sedang diproses, biasanya berlaku pada microsoft word, microsoft powerpoint, notepad, paint, ataupun microsoft excel.

Ctrl + A berfungsi untuk memilih atau select seluruh obyek pada program yang sedang diproses.

Ctrl + X berfungsi untuk melakukan cut pada obyek yang dipilih (memotong obyek, fungsinya hampir sama seperti Ctrl + C namun shortcut ini berfungsi untuk memotong).

Ctrl + V berfungsi untuk melakukan paste obyek yang telah di cut ataupun copy

Jawaban:

1. Fungsi Tombol Esc

Tombol Escape untuk membatalkan perintah yang sudah dilakukkan. Jika kita menekan tombol ini maka perintah atau dialog yang kita masukkan akan dibatalkan.

2. Fungsi Tombol Tab

Tombol Tab berfungsi jika ditekan akan memindahkan cursor ke arah kanan dengan satu tabulansi.

3. Fungsi Tombol Caps Lock 

Jika menekan tombol ini maka lampu indikator caps lock pada keyboard menyala, hal ini menandakan bahwa anda menggunakan mode semua huruf besar atau semua huruf kapital, jika anda menekan lagi dan lampu indikator mati maka dalam mode seperti biasa atau semua huruf kecil.

4. Fungsi Tombol Shift

Fungsinya yaitu untuk membuat huruf besar secara manual (dengan menekan shift+a = A), dan untuk memasukkan input yang diatas angka 1,2,3 dan seterusnya (contoh : shift+1 = ! , shift+/ = ?).

5. Fungsi Tombol Ctrl

Khusus untuk tombol ini fungsinya itu untuk menyisipkan perintah-perintah pintas. Contohnya untuk mencopy  file tinggal tekan ctrl+c.

6. Fungsi Tombol Alt

Fungsinya sama seperti tombol Ctrl yaitu untuk menyisipkan perintah secara pintas. Contohnya alt+F4, itu untuk menutup program yang sedang dijalankan.

7. Fungsi Tombol Enter

Sebagai perintah "ok" yang diberikan oleh program pada komputer dan untuk membuat paragraf baru.

8. Fungsi Tombol Back Space 

Fungsinya untuk mendelete satu huruf yang berada disebelah kiri kursor. Jika kita kombinasikan dengan tombol ctrl, ctrl+backspace maka anda menghapus satu kata disebelah kiri kursor.

9. Tombol Delete

Tombol ini sama seperti tombol backspace, bedanya itu jika tombol delete menghapus satu huruf di sebelah kanan kursor.

10. Tombol Macro 

Macro ini jika ditekan akan muncul seperti ini "\".

11. Tombol Insert 

Untuk menyisipkan karakter di posisi kursor.

12. Tombol Home

Fungsinya untuk langsung menuju bagian atas halaman atau lembar kerja.

13. Tombol End

Fungsinya untuk langsung menuju bagian bawah halaman atau lembar kerja.

14. Tombol Page Up

Fungsinya untuk menggerakkan baris satu demi satu layar ke atas.

15. Tombol Page Down

Fungsinya untuk menggerakkan baris satu demi satu layar ke bawah.

37. sebutkan cara kerja transpose pada keyboard


Jawaban:

Transpose pada keyboard adalah fungsi yang dapat mengubah nada atau key dari sebuah lagu dengan mengubah posisi semua notenya seiring dengan pergeseran nada yang diinginkan.

Cara kerja transpose pada keyboard umumnya dilakukan dengan menggunakan tombol transposisi yang ada di keyboard atau melalui menu transposisi pada layar tampilan. Berikut adalah langkah-langkah umum untuk melakukan transpose pada keyboard:

1. Masuk ke mode transposisi: Pastikan Anda berada dalam mode performa atau mode yang mengizinkan pengaturan transposisi pada keyboard Anda. Biasanya ada tombol "Transpose" atau "Key Transpose" pada keyboard yang dapat digunakan untuk mengakses mode ini. Jika tidak ada tombol khusus, Anda bisa menemukan pengaturan transposisi dalam menu atau tombol fungsi lainnya.

2. Pilih nilai transposisi: Ketika Anda memasuki mode transposisi, Anda akan dapat melihat nilai transposisi saat ini, biasanya dalam bentuk bilangan positif atau negatif. Nilai ini mewakili pergeseran nada dalam setengah nada atau sembilan benda musik. Misalnya, jika Anda ingin meningkatkan kunci dalam lagu sebanyak satu setengah nada, Anda harus mengatur nilai transposisi ke +1. Anda bisa menyesuaikan nilai transposisi sesuai dengan keinginan Anda.

3. Ubah nilai transposisi: Gunakan tombol atau penggerak yang disediakan pada keyboard untuk mengubah nilai transposisi. Pada beberapa keyboard, Anda perlu menekan tombol "+" atau "-" untuk menaikkan atau menurunkan nilai, sementara pada yang lain, Anda dapat menggunakan pengendali berputar atau tombol tertentu yang ditetapkan untuk fungsi ini. Perhatikan bahwa setiap penyesuaian pada nilai transposisi akan mengubah posisi semua notenya secara otomatis.

4. Mainkan lagu dengan transpose: Setelah Anda mengatur nilai transposisi sesuai dengan preferensi Anda, Anda dapat memainkan lagu seperti biasa. Keyboard akan secara otomatis memainkan semua notenya dengan posisi yang dipindahkan sesuai dengan nilai transposisi yang telah Anda tentukan.

Demikianlah cara kerja transpose pada keyboard.

39. jelaskan apa itu transpose pada keyboard


Penjelasan:

Transpose pada keyboard adalah fitur atau fungsi yang memungkinkan kita untuk mengubah kunci nada atau pitch dari nada yang dimainkan tanpa harus mengubah pola fingering atau posisi jari pada keyboard. Dengan fitur ini, kita dapat mengubah nada dasar dari suatu lagu atau mengakomodasi vokal penyanyi yang berbeda tanpa harus memindahkan semua catatan satu per satu.

Misalnya, jika kita memainkan lagu dengan transposisi +2, artinya semua nada yang kita mainkan akan naik dua nada atau dua langkah nada dari not aslinya. Sebaliknya, jika kita menggunakan transposisi -3, maka semua nada akan turun tiga nada atau tiga langkah nada dari not aslinya.

Transpose sangat berguna dalam situasi di mana kita ingin memainkan lagu dalam kunci yang lebih nyaman atau cocok dengan suara kita, atau ketika ingin bermain bersama dengan alat musik lain yang mungkin tidak dapat berubah kunci dengan mudah.

Fitur transpose dapat ditemukan di banyak keyboard elektronik modern dan juga di sebagian besar perangkat lunak musik digital, sehingga memudahkan pemain keyboard untuk menyesuaikan kunci dengan cepat dan mudah.
